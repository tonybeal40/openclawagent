# prompt_library

## Opportunity Ranking Prompt
Rank these opportunities by fit, urgency, and probable response rate. Return top 10 with one-line rationale each.

## Outreach Prompt
Write concise, personalized outreach for the selected top opportunities. Keep CTA clear and low-friction.

## Daily Handoff Prompt
Summarize wins, blockers, and first 3 actions for tomorrow.
