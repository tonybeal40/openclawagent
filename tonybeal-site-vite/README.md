# tonybeal-site-vite

Vite version of tonybeal.net for fast iteration in Replit/VS Code.

## Run locally

```bash
npm install
npm run dev
```

App runs on port 3000.

## Build

```bash
npm run build
npm run preview
```

## Notes
- Static assets are in `public/`.
- Main page is `index.html` + `src/main.js` + `src/style.css`.
- Resume pages are kept in `public/` for direct links.
