# Automation Ideas (Job + Income)

Updated: 2026-03-09

## 1) Job Hunter Agent (daily)
- Find new matching roles in last 48h.
- Output ranked list + direct links + fit reason.

## 2) Company Intelligence Agent
- Scan company pages/news for growth and hiring signals.
- Map likely decision-makers.

## 3) Outreach Draft Agent
- Generate personalized outreach draft from role + company context.

## 4) AI Opportunity Scanner
- Track monetizable trends in AI automation for B2B ops/sales.
- Output practical service ideas and target buyers.

## 5) Weekly Executive Brief
- 1-page report:
  - top job opportunities
  - top target companies
  - outreach stats
  - next-week priorities

## Practical KPI set
- New qualified roles/week
- Outreach sent/week
- Response rate
- Interview rate
- Opportunity-to-offer cycle time
