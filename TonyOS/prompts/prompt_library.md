# TonyOS Prompt Library

## opportunity_ranking
Rank opportunities by fit, urgency, and response likelihood. Return top 10 with one-line rationale and next action.

## company_research
Summarize company: product, size signal, hiring signal, likely pain points, and best contact role.

## outreach_message
Write a concise personalized outreach draft with low-friction CTA and one clear next step.

## daily_briefing
Generate a morning briefing with: top roles, top companies, key signals, and 3 immediate actions.

## job_application
Draft a tailored application summary highlighting relevant wins and role-fit in under 150 words.
