# Mission Control (JS)

Lightweight JavaScript Mission Control inspired by the transcript you shared.

## Screens included
- Task Board (Kanban + activity)
- Calendar (scheduled jobs)
- Projects (progress + priority)
- Memories (daily + long-term)
- Docs (searchable)
- Team (roles + mission)
- Office (fun activity wall)

## Run
1) Export workspace data into dashboard JSON:

```powershell
cd projects/mission-control-js
powershell -ExecutionPolicy Bypass -File scripts/export-data.ps1
```

2) Serve this folder:

```powershell
python -m http.server 8080
```

3) Open `http://localhost:8080`.

## Data
The app loads JSON from `data/*.json` (generated by `scripts/export-data.ps1`) and falls back to localStorage seed data.

## Current integrations
- `memory/*.md` -> `data/memories.json`
- `ops/WORKER_ROSTER.md` -> `data/team.json`
- `runbooks/`, `outputs/`, `memory/` docs index -> `data/docs.json`
- Best-effort cron snapshot -> `data/jobs.json`
